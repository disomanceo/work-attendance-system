แก้ฟอนต์และเลขไทยใน PDF รายวัน

สิ่งที่ปรับ:
- ใช้ฟอนต์ Sarabun ทั้งเอกสาร
- น้ำหนัก Normal สำหรับเนื้อหา
- หัวเรื่อง 16 pt ตัวหนา
- ข้อความทั่วไป 14 pt Normal
- หัวตาราง 14 pt ตัวหนา
- ข้อมูลในตาราง 14 pt Normal
- แปลงเลขทั้งหมดเป็นเลขไทย:
  ลำดับ / วันที่ / ปี / เวลา / หมายเหตุ / จำนวนสรุป
- ลดปัญหาตัวหนังสือเล็กใหญ่ไม่เท่ากัน
- กำหนดระยะก่อน-หลังย่อหน้าเป็น 0
- กำหนดระยะบรรทัดเป็น 1

ไฟล์ที่แก้:
gas/Code.gs

ติดตั้ง:
1. นำ gas/Code.gs ไปแทนไฟล์ Code.gs เดิมใน Google Apps Script
2. Save
3. Deploy > Manage deployments
4. Edit > New version > Deploy
5. กลับหน้าเว็บแล้วกด “สร้าง PDF ใหม่”

ไม่ต้องแก้ไฟล์ Next.js และไม่ต้องแก้ .env.local
